
## Communication

- 永远使用简体中文进行思考和对话

## Documentation

- 正式文档写到项目的 docs/ 目录下
- 用于讨论和评审的计划、方案等文档，写到项目的 docs/discuss/ 目录下
- 项目的根目录必须保持简洁，只保留必须存在的文件

## Code Architecture

- 用 python 语言在 pipeline/ 目录下构建程序执行的 pipeline
- 算法利用 C++ 语言在 pnc/ 目录下实现
- python 语言不用于实现算法，请调用 C++ 算法库下相关算法
