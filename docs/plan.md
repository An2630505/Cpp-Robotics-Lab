## 下一步计划

当前我们已经有一个比较基础的demo实现了，实现静态障碍物下的轨迹避障和路径规划；

下面，我们希望构建更为真实和合理的场景与测试，构建更为系统的场景实现

我们的目标是希望实现一套完整的自动驾驶闭环框架系统：

包含：

数据 -> 算法 -> 回灌（IDE） -> 仿真（engine）

我们并不追求造轮子，但是不希望简单的运行和使用他人的成果，我们希望在参考他人优秀作品的基础上，通过比较深入的重构和结合，来实现自我的一套系统；


我们当前已经完成了一套比较简单的 算法demo，所以我们后续的任务是去完成相应的IDE和仿真引擎，将IDE和仿真还有算法剥离开来：

1. 简单实现了一个2D层面的运动学仿真引擎；
2. 实现IDE设计；
